# My Portfolio

## 📌 Project Overview
This is a personal portfolio website built using basic HTML, CSS, and JavaScript to showcase projects, skills, and contact information.

---
## 🚀 Features
- Clean and simple UI
- Responsive design
- Projects showcase section
- About me section
- Contact information

---
## 🧠 Technologies Used
- HTML
- CSS
- JavaScript

---
## ⚙️ How It Works
1. Structure is created using HTML
2. Styling is done with CSS
3. Interactivity is added using JavaScript
4. Content displays projects and personal details
